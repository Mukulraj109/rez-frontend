## Best Practices

### âœ… DO
- Use optimistic updates for instant feedback
- Show loading states during API calls
- Display helpful error messages
- Keep comment input sticky at bottom
- Auto-focus input when replying
- Show character counter near limit
- Use pull-to-refresh for manual refresh
- Implement infinite scroll for pagination

### âŒ DON'T
- Block UI during API calls
- Hide errors from users
- Allow comments over 500 chars
- Nest replies beyond 1 level
- Show stale comment counts
- Forget to handle keyboard

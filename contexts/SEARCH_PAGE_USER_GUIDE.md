## ğŸš€ How to Use

### Basic Search

1. **Open Search Page**
   - Tap the search icon in the navigation bar
   - Or tap the search bar on the home page

2. **Enter Search Query**
   - Type what you're looking for (min 2 characters)
   - Results appear automatically after you stop typing

3. **Browse Results**
   - Scroll through products and stores
   - Tap any result to view details

### Category Browsing

1. **View Categories**
   - Open the search page
   - Browse featured categories at the top

2. **Select a Category**
   - Tap any category card
   - See all items in that category
   - Filter and sort as needed

### Using Filters

1. **Open Filter Modal**
   - Search for something
   - Tap the "Filter" button at the top

2. **Set Filters**
   - **Price Range**: Drag sliders to set min/max price
   - **Rating**: Tap to select minimum rating
   - **Category**: Choose one or more categories

3. **Apply Filters**
   - Tap "Apply Filters" button
   - Results update automatically
   - Filter count shows in the button

4. **Clear Filters**
   - Tap "Clear All" in the filter modal
   - Or tap the "X" next to individual filters

### Sorting Results

1. **Open Sort Modal**
   - Search for something
   - Tap the "Sort" button

2. **Select Sort Option**
   - Tap any sorting option
   - Results reorder automatically
   - Current sort shows in the button

### Search History

1. **View History**
   - Open search page
   - Scroll down to see "Recent Searches"

2. **Repeat a Search**
   - Tap any recent search
   - Search executes immediately

3. **Clear History**
   - Tap "Clear All" button
   - Confirm in the dialog

## ğŸ’¡ Tips & Tricks

### Search Tips

1. **Be Specific**
   - Instead of "food", try "pizza delivery"
   - Instead of "shop", try "electronics store"

2. **Use Keywords**
   - Include brand names: "Samsung phone"
   - Include location: "salon near me"
   - Include features: "wireless headphones"

3. **Check Spelling**
   - The search is smart but works best with correct spelling
   - Use auto-complete suggestions when available

4. **Try Different Terms**
   - If no results, try synonyms
   - "Restaurant" vs "Diner" vs "Eatery"

### Filter Tips

1. **Start Broad, Then Narrow**
   - Search first, then apply filters
   - Use fewer filters for more results

2. **Combine Filters**
   - Price + Rating for best value
   - Category + Location for specific needs

3. **Save Favorite Filters**
   - Use history to repeat successful searches
   - Bookmark frequently used filter combinations

### Performance Tips

1. **Use Cache**
   - Repeated searches load instantly
   - Cache lasts 5 minutes

2. **Limit Results**
   - Use specific search terms
   - Apply filters early

3. **Stable Connection**
   - Search works better with WiFi
   - Offline searches use cache

## â“ Frequently Asked Questions

### General Questions

**Q: How do I search for multiple things?**  
A: Search for one thing at a time for best results. Use filters to narrow down.

**Q: Can I search by voice?**  
A: Voice search is coming soon! For now, use the keyboard.

**Q: Why are some results shown first?**  
A: Results are sorted by relevance, but you can change this using the Sort button.

**Q: Can I search for specific stores?**  
A: Yes! Search by store name, and it will appear in the results.

### Search Issues

**Q: No results found. What should I do?**  
A: Try these steps:
1. Check spelling
2. Use different keywords
3. Remove filters
4. Try a broader search term

**Q: Search is slow. How to fix?**  
A: 
1. Check your internet connection
2. Close and reopen the app
3. Clear search history
4. Update the app to latest version

**Q: Results are not relevant. Why?**  
A:
1. Be more specific in your search
2. Use filters to narrow down
3. Try exact product/store names

### Filter & Sort Issues

**Q: Filters are not working. What to do?**  
A:
1. Make sure you clicked "Apply Filters"
2. Try clearing all filters and reapplying
3. Refresh the search

**Q: Can I save my filter settings?**  
A: Not yet, but you can use search history to repeat filtered searches.

**Q: How to remove a single filter?**  
A: Tap the "X" next to the filter name, or open the filter modal and adjust.

### History & Privacy

**Q: How to delete search history?**  
A: Tap "Clear All" in the Recent Searches section.

**Q: Is my search history private?**  
A: Yes, it's stored only on your device. We don't share it.

**Q: How long is history kept?**  
A: Last 10 searches are kept until you clear them or uninstall the app.
